# Forecast_UMM_ENSO.py (Forecast_UMM_ENSO_IPO.py)

Predicts ENSO (ENSO/IPO) states, rainfall categories and replicate rainfall categories. 

## Documentation

Evaluates the Brier Skill Scores (BSS) for predicted ENSO (ENSO/IPO) states and rainfall categories.

# HMM_Predict_states_ENSO.py (HMM_Predict_states_ENSO_IPO.py)

Predicts ENSO (ENSO/IPO) and plots Reliability and Relative Operating Characteristics (ROC) attribute diagrams.

## Documentation

Reliability and ROC attribute diagrams evaluate forecast reliability and quality of probabilistic forecast system respectively.  

# ROC_mapping.py

Produces ROC scores for each ENSO (ENSO/IPO) state.

## Documentation

ROC scores evaluate the quality of ENSO (ENSO/IPO) state forecasts.

# SPOTA_Like_Forecast_ENSO.py (SPOTA_Like_Forecast_ENSO_IPO.py)

Produces probabilistic seasonal rainfall outlooks for each rainfall category.

## Documentation

Plots probabilistic seasonal rainfall outlook pie charts based on the projected ENSO (ENSO/IPO) state.

# DF_Aggregate.py, Forecast_Common.py, Forecast_Models.py, Forecasts_Plots.py 

File IO functions for numpy and pandas utilised by all the scripts above.

# QLD_Apr_Mar_SOI_Jun_Mar_TPI_Jul_Jun.csv

Data file.

## Documentation

File contains Queensland April-to-March annual rainfall, Jun-to-March average SOI, and July-to-June average Tripole Index (TPI) which represents IPO.
